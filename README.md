# SQUADFLOW

**AI Development Team · Operating System**

> The platform for software companies to build, deploy and manage using AI agent squads.

## Structure

```
SQUADFLOW/
├── apps/
│   ├── portal/          ← SaaS Portal (Angular)
│   └── admin/           ← Platform Admin (Angular)
├── modules/
│   ├── crm/             ← CRM Module
│   ├── tickets/         ← Ticket Management Module
│   └── billing/         ← Billing Module (coming soon)
├── backend/
│   ├── api/             ← Node.js REST API
│   └── db/              ← MS-SQL Schemas & Migrations
├── shared/
│   ├── components/      ← Shared Angular Components
│   ├── models/          ← TypeScript Interfaces
│   └── services/        ← Shared Services
├── docs/                ← Documentation
└── tools/               ← Scripts & Utilities
```

## Tech Stack

- **Frontend:** Angular + PrimeNG UI Kit
- **Backend:** Node.js + Express
- **Database:** MS-SQL Server
- **Auth:** JWT + Multi-tenant
- **AI:** SQUADFLOW Agent Methodology
